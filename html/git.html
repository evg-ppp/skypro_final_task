Git

  • git init - расшифровывается,как "инициализация нового репозитория".
  • git add - расшифровывается,как "добавление изменений в индекс".
  • git commit - расшифровывается,как "сохранение изменений".
  • git push- расшифровывается,как "отправка изменений в удалённый репозиторий".
  • git pull- расшифровывается,как "получение изменений из удалённого репозитория".
